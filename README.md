# homebrew-tap

Homebrew tap hosting formulas for multiple projects.

## Install

```bash
brew install flaviusmatis/tap/<formula>
```

## Available formulas

- **smartcard** — Smartcard signing service with live activity TUI.

  ```bash
  brew install flaviusmatis/tap/smartcard
  smartcard                        # interactive TUI
  brew services start smartcard    # background daemon, autostart at login
  ```
